# 🌴 Bali 2026 — Piano di Viaggio

Sito web del piano di viaggio per Bali, Aprile–Maggio 2026.

## File principali
- `index.html` — Piano completo del viaggio (bali-piano-finale.html)
- `mappa.html` — Mappa interattiva con tutti i punti di interesse

## Come aggiornare il sito
1. Modifica i file HTML
2. Salva
3. Dal terminale, nella cartella `web/`:
```
git add .
git commit -m "aggiornamento"
git push
```
Netlify fa il deploy automaticamente in ~30 secondi.
